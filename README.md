# 2026 융합전공 소개행사 — 시스템 운영 매뉴얼

**행사 일시:** 2026년 5월 8일(금) 09:30 ~ 17:00
**장소:** 한양종합기술원(HIT) 1층 양민용 커리어라운지
**발신 이메일:** intercollege@hanyang.ac.kr

---

## 이 시스템은 무엇인가요?

학생들의 **사전 예약 접수부터 당일 체크인, 기념품 수령까지** 전 과정을 온라인으로 처리하는 웹 기반 관리 시스템입니다. Google Apps Script(구글 스프레드시트 기반)로 만들어졌으며, 별도 설치 없이 크롬 브라우저에서 접속해 사용합니다.

---

## 페이지 종류 및 접속 방법

| 페이지 | 접속 주소 | 비밀번호 | 사용 대상 | 주요 기능 |
|---|---|---|---|---|
| **예약 페이지** | https://tinyurl.com/hicmajors-booking | — | 학생 | 부스 상담 예약, 설명회 사전예약, 예약 변경·취소 |
| **스태프 페이지** | https://tinyurl.com/hicmajor-staff | hic2025 | 스태프 | 현장 체크인 처리, 기념품 수령 처리 |
| **관리자 페이지** | https://tinyurl.com/hicmajors-dashboard | hic2025 | 행정팀·교수자 | 통계 확인, 참석자 명단 출력, 부스 관리, 확인서 발급 |

> 비밀번호는 AdminUsers 시트에 등록된 계정 기준입니다. **행정팀 로그인은 담당 프로그램: 전체관리**를 선택하세요.

### 전체 참석 현황 (대시보드)

로그인하면 상단에 자동으로 통계가 나타납니다.

| 항목 | 설명 |
|---|---|
| **설명회 참석 건수** | 1명이 3개 설명회에 참석하면 3건으로 계산 |
| **설명회 참석 인원** | 학번 기준 중복 제거 — 실제 몇 명이 참석했는지 |
| **사전예약 참석 건수/인원** | 미리 예약하고 온 학생 |
| **당일 참석 건수/인원** | 예약 없이 현장에서 입장한 학생 |
| **기념품 수령 인원** | 기념품을 수령하고 서명한 학생 수 |

> 처음 로딩은 서버가 켜지는 데 최대 15초가 걸릴 수 있습니다. 🔄 새로고침 버튼으로 최신 데이터를 불러오세요.

---

### 설명회 체크인 탭 (✍️)

참석자 명단 확인 및 PDF 출력에 사용합니다.

1. 확인할 **설명회 이름**을 클릭
2. 참석자 목록, 체크인 시각, 사전/당일 구분이 표시됨
3. **참석자 명단 PDF** 버튼으로 출력 (예산 증빙 및 보관용)

---

### 부스 예약 탭 (📋)

부스별 예약 현황 확인 및 슬롯 차단에 사용합니다.

1. **프로그램(부스) 이름**을 클릭
2. 시간대별 예약자 명단이 표시됨
3. 빈 슬롯을 클릭해 **차단/해제** 가능 (담당자 자리 비움 등 상황에서 사용)
4. 예약자 메모 아이콘을 클릭하면 학생의 문의 내용 확인 가능

> **현장예약 마감 버튼**: 상단에 위치. 누르면 신규 예약이 전면 중단됩니다. 행사 당일 예약 마감 시 사용하세요. 같은 버튼으로 재개도 가능합니다.

---

### 참여확인서 탭 (🎁)

학생 개별 참여확인서 발급 및 기념품 서명부 출력에 사용합니다.

1. 학번 입력 → 참여 내역 조회
2. **PDF 저장**: 확인서를 PDF로 출력
3. **이메일 발송**: 학생 이메일로 확인서 직접 발송
4. **기념품 서명부 PDF**: 전체 수령자 명단 출력 (예산 증빙용)

## 스태프 페이지 사용법

스태프 계정(AdminUsers 시트에 등록된 계정)으로 로그인합니다.

### 체크인 모드

**사전예약 학생**
1. 학번 입력 또는 바코드 스캔
2. 예약 내역 확인 후 **체크인** 버튼 클릭

**당일방문 학생**
1. 학번 + 한양 이메일(@hanyang.ac.kr) 입력
2. **현장입장** 버튼 클릭
3. 현재 세션 이후의 모든 설명회가 자동으로 참석 처리됨

---

### 세션 진행 관리

설명회가 끝날 때마다 스태프가 직접 세션을 넘겨야 합니다.

1. 우상단 **다음 세션 →** 버튼 클릭
2. 비밀번호 입력
3. 다음 설명회로 진행

> 세션 순서와 시간은 Settings 시트의 `설명회세션` 행에서 자동으로 불러옵니다.
> 잘못 눌렀을 경우 같은 버튼에서 이전 세션으로 되돌릴 수 있습니다.

---

### 기념품 수령 모드

1. 학번 입력 → 참여 내역 및 스탬프 수 확인
2. 수령할 스탬프 체크 (3개 이상 필요)
3. 인터칼리지 학생은 가산점 과목 선택 필수
4. 서명란에 서명 후 **수령 완료** 처리

> 라이프디자인센터(LDC) 스탬프는 **한양인터칼리지학부 재학생만** 선택 가능합니다.

## Settings 시트 구조

Settings 시트에 아래 정보를 입력하면 예약 페이지·스태프 페이지·관리자 페이지 전체에 자동으로 반영됩니다. **행사 정보나 프로그램이 바뀔 때마다 이 시트만 수정하면 됩니다.**

| A열 (항목) | B열 (값 / 프로그램명) | C열 (시작시간) | D열 (종료시간) |
|---|---|---|---|
| `예약오픈여부` | `OPEN` 또는 `CLOSED` | | |
| `행사명` | 행사 부제목 문자열 | | |
| `행사일시` | 날짜·시간 문자열 | | |
| `행사장소` | 장소 문자열 | | |
| `currentSessionIdx` | `0` (행사 시작 전 초기값) | | |
| `설명회세션` | 프로그램명 | `HH:MM` | `HH:MM` |
| `설명회세션` | … (순서대로 반복) | | |
| `부스프로그램` | 프로그램명 | | |
| `부스프로그램` | … (순서대로 반복) | | |

**규칙**
- `설명회세션` 행은 **위에서부터 순서대로** 1단계, 2단계… 로 처리됩니다.
- `부스프로그램` 행은 예약 페이지와 기념품 수령 화면에 표시되는 목록입니다.
- `라이프디자인센터`처럼 설명회는 없고 부스만 있는 프로그램은 `부스프로그램`에만 추가하면 됩니다.
- `currentSessionIdx`는 행사 당일 스태프가 직접 진행시키는 값이므로 **행사 전날 반드시 0으로 초기화**하세요.

## 다음 행사를 위한 재설정 방법

매년 행사 준비 시 아래 항목만 수정하면 시스템 전체가 새 정보로 동작합니다.

### 1. Settings 시트 수정
- `행사명`, `행사일시`, `행사장소` 값 변경
- `설명회세션` 행: 프로그램명·시작시간·종료시간 수정 (행 추가/삭제 가능)
- `부스프로그램` 행: 부스 목록 수정 (행 추가/삭제 가능)
- `currentSessionIdx` → `0` 으로 초기화

### 2. AdminUsers 시트 수정
- 스태프 계정 비밀번호 변경
- 담당 교수자 계정 추가/수정

### 3. 리마인드 메일 트리거 재설정
- Apps Script 편집기에서 `setupRemindTrigger()` 함수를 실행하여 발송 일정을 새 행사 일정에 맞게 재등록

### 4. 예약 오픈
- Settings 시트의 `예약오픈여부`를 `OPEN`으로 변경

> **코드 수정은 필요 없습니다.** 모든 프로그램 정보와 행사 정보는 Settings 시트에서 불러옵니다.

## 자주 묻는 질문

**Q. 통계 숫자가 0으로 보여요**
→ 🔄 새로고침 버튼을 누르거나 페이지를 다시 로그인해 보세요. 처음 로딩은 최대 15초가 걸릴 수 있습니다.

**Q. 부스 슬롯을 잘못 차단했어요**
→ 부스 예약 탭에서 같은 슬롯을 다시 클릭하면 차단이 해제됩니다.

**Q. 참여확인서 PDF가 안 열려요**
→ 브라우저의 팝업 차단 설정이 켜져 있을 수 있습니다. 주소창 오른쪽의 팝업 허용 버튼을 눌러주세요.

**Q. 학생이 예약을 취소했는데 명단에 남아있어요**
→ 취소된 예약은 시트에서 자동 삭제됩니다. 새로고침 후에도 보인다면 담당 개발자에게 문의하세요.

**Q. 리마인드 메일이 안 갔어요**
→ Apps Script 편집기에서 `setupRemindTrigger()` 함수가 실행되어 있는지 확인하세요. (최초 1회 실행 필요)

**Q. 내년에도 이 시스템을 쓰려면 어떻게 하나요?**
→ Settings 시트의 행사 정보와 세션·부스 목록만 수정하면 됩니다. 코드 수정은 필요 없습니다. 위 **다음 행사를 위한 재설정 방법** 항목을 참고하세요.
